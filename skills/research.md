# Research & Comparison

## Khi nghiên cứu library/tool mới
1. **Problem**: Vấn đề cần giải quyết?
2. **Candidates**: 3-5 options phổ biến
3. **Criteria**: Stars, last commit, performance, API, license
4. **Recommendation**: Chọn 1, giải thích tại sao
5. **Confidence**: Mức chắc chắn

## Output Format
```
### [Topic]
**Vấn đề**: ...
| Criteria | Option A | Option B | Option C |
|----------|----------|----------|----------|
| ...      | ...      | ...      | ...      |
**Đề xuất**: Option X vì...
**Nguồn**: [link]
```

## Khi không chắc chắn
- Nói rõ: "Em chưa chắc lắm về phần này"
- Đề xuất verify thêm ở nguồn cụ thể
- Phân biệt: fact vs opinion vs guess
- Web search khi cần info mới
